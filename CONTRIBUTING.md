# Contributing

Púca is maintained as a personal open-source project.

External pull requests are not accepted at this time. This keeps project
direction, licensing, and copyright ownership simple while the project moves
from private development to a public repository.

Issues and bug reports are welcome. When reporting a problem, please include:

- What you expected to happen.
- What actually happened.
- The route, stop, or train search involved, if relevant.
- Browser and device details, especially for map, install, or geolocation bugs.
- Screenshots or logs when they help explain the issue.
